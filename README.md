# Pandemic-Flight-Analysis-USA
Analyse de l'impact du COVID-19 sur les vols aux USA via Tableau (Jan-Avr 2020)

#  Analyse d'Impact Pandémique : Vols USA (Jan-Avr 2020)

##  Présentation
Ce projet analyse les perturbations majeures du trafic aérien américain durant le début de la pandémie de COVID-19. L'objectif est d'identifier les zones géographiques les plus touchées et d'analyser la résilience des compagnies aériennes.

##  Fonctionnalités du Dashboard Tableau
* **Map Interactive** : Visualisation géographique des aéroports avec les plus gros taux de retards/annulations.
* **Heatmaps temporelles** : Corrélation entre la montée des cas COVID et la chute du trafic.
* **Performance Compagnies** : Analyse comparative des délais via des **Stacked Bar Charts**.
* **Analyse des causes** : Répartition des retards (Météo, Sécurité, COVID-19, Retard transporteur).


##  Compétences démontrées
* Storytelling avec les données.
* Création de calculs complexes sous Tableau.
* Design de dashboards orientés "Décideurs".
